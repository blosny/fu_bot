obs anket botu

okul sistemindeki zorunlu anketleri doldurmaya üşenenler için yazdık. çalıştırıyorsunuz kendi kendine arka planda doldurup geçiyor. 

kurulum:
1- terminale şunu yazın: pip install -r requirements.txt
(hata verirse elle pip install selenium ve pip install webdriver-manager yazın)
2- main.py dosyasını açıp ogrenci_no ve sifre kısımlarına kendi bilgilerinizi yazın.
3- python main.py ile çalıştırın.
4- galiba sorunsuz çalışacak.

dikkat: kodları github'a falan atarken veya biriyile paylaşırken kendi şifrenizi silmeyi unutmayın.
